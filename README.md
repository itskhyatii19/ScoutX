# ScoutX 🏆

ScoutX is an AI-powered athlete scouting and networking platform designed to connect athletes, coaches, and organizations worldwide.

It helps users:

* Discover tournaments 🌍
* Analyze performance 📈
* Network with professionals 🤝
* Get AI-driven insights 🤖
* Upload match/training videos 🎥

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/itskhyatii19/ScoutX.git
cd ScoutX
```

### Frontend Setup

```bash
cd frontend
npm install
npx expo start
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```
